Тест-кейсы для API микросервиса объявлений

TC-001: Успешное создание объявления
Приоритет: high
Шаги:
1. Отправить POST запрос на /create с валидными данными
2. Проверить статус код 200
3. Проверить наличие item_id в ответе

TC-002: Создание без обязательных полей
Приоритет: high
Шаги:
1. Отправить POST запрос без поля sellerId
2. Проверить статус код 400
